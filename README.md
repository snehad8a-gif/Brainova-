\## ✨ Brainova



A modern, gamified AI-powered study habit tracking app built with Streamlit. \*\*Brainova\*\* helps you build and maintain positive academic habits, track your progress, and stay motivated with smart suggestions and analytics.



\---



\### 🚀 Features



\* \*\*📝 Reminders:\*\* Set and manage sticky reminders for non-habit tasks (e.g., "Submit Assignment", "Review Lecture Notes").

\* \*\*📅 Daily Focus:\*\* See your study habits for today, mark them as done, and get a sense of accomplishment.

\* \*\*💡 Smart Suggestions:\*\* AI-driven recommendations to help you stay on track and improve your study routines.

\* \*\*📊 Analytics:\*\* Visualize your habit streaks, completion rates, and academic progress over time.

\* \*\*⚙️ Settings:\*\* Edit or delete habits, manage your data, and customize your experience.

\* \*\*🔔 Priority Levels:\*\* Color-coded reminders for high, medium, and low priority tasks.

\* \*\*🎨 Beautiful UI:\*\* Custom CSS for a modern, gamified, and visually appealing experience.



\---



\### 🛠️ Tech Stack



\* \*\*Streamlit\*\* for the interactive web UI

\* \*\*Pandas\*\* for data management

\* \*\*Custom Python modules\*\* for database, analytics, ML logic, and UI components

\### 🏁 Getting Started



1\. \*\*Clone the repository\*\*

```bash

cd brainova



```





2\. \*\*Install dependencies\*\*

```bash

pip install -r requirements.txt



```





3\. \*\*Run the app\*\*

```bash

streamlit run app.py



```
